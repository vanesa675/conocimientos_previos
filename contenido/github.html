Github


Comandos utilizados



git config --global user.name "vanesa": Se utiliza para configurar el nombre que sale en los commits.
git config --global user.email lgutierrezlozano09@gmail.com: Se utiliza para configurar el email.

iniciando repositorio



git init: Se utiliza para iniciar git en la carpeta donde esta el proyecto.
git clone: Se utiliza para clonar el repositorio de Github.
git add .: Se utiliza para agregar los archivos para realizar el commit.
git commit -m "comentario": Se utiliza para comentar.
git push: Se utiliza para subir los archivos al repositorio.

Git Log



git log: Se utiliza para rostrar los logs de los commits.
git log --oneline --stat: Se utiliza para mostrar los cambios en los commits.
