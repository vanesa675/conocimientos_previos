HTML


Las etiquetas semánticas en HTML5 son elementos que proveen información sobre el significado y la función del contenido que encapsulan. A diferencia de las etiquetas no semánticas que solo controlan la presentación visual, las etiquetas semánticas aportan información contextual a los navegadores, lectores de pantalla y otros agentes de software.



tipos de etiquetas semánticas:

Etiquetas de texto:

h1 a h6: Definen encabezados de diferentes niveles.
p: Define un párrafo de texto.
var: Define una variable.

Etiquetas de agrupación:

header: Define el encabezado de una sección.
article: Define un artículo independiente.
section: Define una sección genérica de contenido.

Etiquetas multimedia:

img: Inserta una imagen.
video: Inserta un video.
audio: Inserta un audio.
figure: Agrupa contenido multimedia con su leyenda.

Etiquetas de tablas:

table: Define una tabla.
tr: Define una fila de una tabla.
td: Define una celda de una tabla.
thead: Define el encabezado de una tabla.
tbody: Define el cuerpo de una tabla.

Etiquetas de formularios:

form: Define un formulario.
input: Inserta un campo de entrada de datos.
label: Define una etiqueta para un campo de entrada.
select: Define un menú desplegable.
option: Define una opción dentro de un menú desplegable.
button: Define un botón.
