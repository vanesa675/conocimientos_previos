CSS


Las hojas de estilo en cascada, o CSS3 son un lenguaje de programación que se utiliza para controlar el aspecto y la presentación de las páginas web.



Tipos de selectores



Selectores por tipo: Seleccionan elementos por su nombre de etiqueta (ej. h1, p).
Selectores por clase: Seleccionan elementos que tienen una clase específica (ej. .clase).

Selectores por ID:Seleccionan un único elemento con un ID específico (ej. #id).
Selectores descendientes: Seleccionan elementos que son descendientes de otro elemento (ej. div p).
Selectores de pseudo clases: Seleccionan elementos en función de su estado (ej. :hover).

Pseudo clases



:hover: Se aplica cuando el cursor se encuentra sobre un elemento.
:active: Se aplica cuando se hace clic en un elemento.

Unidades de medida relativas



em: Un porcentaje del tamaño de fuente del elemento padre.
rem: Un porcentaje del tamaño de fuente del elemento raíz.
Vw: Un porcentaje del ancho de la ventana gráfica.
Vh: Un porcentaje del alto de la ventana gráfica.

Fondos y bordes



Fondos: Se pueden usar imágenes, colores o degradados para rellenar el fondo de un elemento.
Bordes: Se pueden usar para agregar un borde alrededor de un elemento.

Modelo de cajas



Content-box: El tamaño del contenido del elemento.
Padding-box: El tamaño del contenido más el relleno.
Border-box: El tamaño del contenido, el relleno y el borde.

Position



Static: El elemento se coloca en su posición normal en el flujo del documento.
Relative: El elemento se coloca en su posición normal, pero puede ser desplazado con respecto a ella.
Absolute: El elemento se coloca en una posición específica con respecto a su elemento padre posicionado más cercano.

Display



Block: El elemento se muestra como un bloque rectangular.
Inline: El elemento se muestra en línea con el texto adyacente.
none: El elemento no se muestra.

Responsive Design “Media Queries”



Permiten que el diseño de una página web se adapte a diferentes dispositivos y tamaños de pantalla.
Se pueden usar para cambiar el tamaño de fuente, el diseño de la página y la visibilidad de los elementos.
